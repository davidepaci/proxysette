WAV-PRG 4.2.1
-------------
This archive contains:
* wav2prg.exe, converts Commodore tapes into T64, PRG and P00
* prg2wav.exe, creates Commodore tapes from T64, PRG and P00, and lists
  contents of T64, PRG and P00

Those are command-line tools: they are meant to be run from a command prompt.
They need audiotap.dll, and wav2prg.exe needs the plug-ins. Therefore, unzip
these file in the same folder where wavprg.exe and the accessory files are.
